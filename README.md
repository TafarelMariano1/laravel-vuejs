# Laravel + Vuejs = s2

Design created to test Laravel with VueJS.

* Laravel 5.5
* VueJS 2

## Commands Laravel

Create auth Laravel

```unix
php artisan make:auth
```

Migrate tables

```unix
php artisan migrate
```

Create controller

```unix
php artisan make:controller Admin/NameController --resource
```
